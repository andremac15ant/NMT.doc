Política de Privacidade

Última atualização: 07 de janeiro de 2026

1. Introdução

Esta Política de Privacidade descreve como o bot [Nome do Bot] coleta, utiliza e protege informações necessárias para seu funcionamento.

Ao utilizar o Bot, você concorda com esta Política.

2. Dados Coletados

O Bot pode coletar e processar as seguintes informações:

Dados do Discord
IDs de usuários;
IDs de servidores;
IDs de canais;
IDs de cargos;
IDs de mensagens.
Dados Operacionais
Comandos executados;
Configurações do servidor;
Informações necessárias para funcionamento dos recursos do Bot.
3. Dados Não Coletados

O Bot não coleta intencionalmente:

Senhas;
Dados bancários;
Informações de pagamento;
Mensagens privadas, exceto quando necessário para execução de funcionalidades específicas iniciadas pelo usuário.
4. Finalidade do Tratamento

Os dados processados são utilizados exclusivamente para:

Executar comandos solicitados pelos usuários;
Fornecer funcionalidades do Bot;
Manter configurações dos servidores;
Corrigir erros e melhorar o desempenho do serviço;
Garantir a segurança e estabilidade da aplicação.
5. Compartilhamento de Dados

Não vendemos, alugamos ou compartilhamos dados pessoais com terceiros.

Os dados poderão ser divulgados apenas quando:

Exigido por lei;
Necessário para cumprimento de obrigações legais;
Necessário para proteger a segurança da aplicação ou de seus usuários.
6. Armazenamento e Retenção

Os dados serão armazenados apenas pelo período necessário para o funcionamento do Bot.

O Desenvolvedor poderá excluir dados a qualquer momento sem aviso prévio.

7. Segurança

São adotadas medidas técnicas e organizacionais razoáveis para proteger as informações processadas contra:

Acesso não autorizado;
Alteração indevida;
Divulgação não autorizada;
Perda de dados.

Entretanto, nenhum sistema é completamente seguro.

8. Direitos dos Usuários

Os usuários podem solicitar:

Informações sobre dados armazenados;
Correção de informações incorretas;
Exclusão de dados, quando aplicável.

Solicitações podem ser feitas através dos canais de contato informados nesta Política.

9. Serviços de Terceiros

O Bot depende da infraestrutura fornecida por:

Discord
GitHub (caso utilizado)
Serviços de hospedagem utilizados pelo Desenvolvedor

O tratamento de dados realizado por esses serviços é regido por suas respectivas políticas de privacidade.

10. Alterações desta Política

Esta Política poderá ser modificada a qualquer momento.

A versão mais recente estará sempre disponível nesta página.
